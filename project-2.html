Personal Dashboard

A Chrome Extension that gives you an overview over the current time, how the weather is like and how Dogecoin is performing. Interacts with multiple APIs using async JS.

Live Link

Project Overview

In an increasingly digital world, staying updated with the latest information is essential. This Personal Dashboard offers a streamlined, interactive interface that aggregates key data points—from live cryptocurrency prices to localized weather conditions—while showcasing dynamic background imagery that refreshes with every visit. It’s designed to empower users with real-time insights, ensuring that essential information is always just a glance away.

Built with modern web technologies and reliable APIs, this dashboard not only delivers accurate, timely data but also provides a visually engaging experience tailored to your daily needs. Whether you’re tracking digital currency trends, checking the current weather, or simply enjoying a new aesthetic with each load, this tool transforms routine updates into an inspiring, personalized online experience.
